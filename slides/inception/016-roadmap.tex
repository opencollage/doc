\section{Roadmap}

\begin{frame}{Roadmap: From Spec to Ecosystem}
  \begin{block}{Core Message}
    \scriptsize
    A phased approach---spec first, reference implementation second,
    ecosystem third---ensures the standard is solid before adoption scales.
  \end{block}
